# AI Engineering Playbook

A centralized repository for managing AI engineering agents, prompts, coding standards, reusable instructions, and workflows across multiple AI tools such as:

- GitHub Copilot
- Claude
- ChatGPT
- VS Code AI Agents
- Future AI development assistants

This repository is designed to support professional software engineering workflows with a strong focus on:

- Production-quality code
- Clean Architecture
- TDD (Test Driven Development)
- Maintainability
- Security
- Scalability
- AI-assisted development standards
